Vision 4 print


Introduction:
This project involved redesigning the website for Vision4Print, a printing services company, in order to modernize the aesthetics, amplify the brand, and enhance user experience.

User Research:
To identify areas for improvement, I first conducted stakeholder interviews, reviewed web analytics, and analyzed competitor sites. Key issues found included outdated visuals, cluttered layouts, and weak brand expression.

UX Design:
I started by optimizing the information architecture and creating new user flows tailored to customer journeys based on research findings. This provided an improved navigational structure.

Visual Design:
For the visual redesign, I explored style tile concepts focused on minimalist, sophisticated aesthetics that better conveyed the brand essence. I refined typography, color scheme, spacing, imagery, and created high-fidelity visual comps in Figma.

Results:
The redesigned Vision4Print website featured a streamlined, uncluttered interface with refined typography and ample negative space.
